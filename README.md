# **TODO Application**

A simple and interactive todo application to store all the task to be done.\
It has the following features:

> * **Add Task**
> * **Edit Task**
> * **Delete Task**

## **Running the app**

To run the app, in the command line/terminal type:\
`node app.js`\
OR after installing nodemon\
`nodemon app.js`
